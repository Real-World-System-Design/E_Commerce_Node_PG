## Technologies used :
* NodeJS     -- Platform
* Express    -- Server
* TypeScript -- Language
* PostgreSQL -- DataBase
* TypeORM    -- ORM
* Bcrypt     -- Hashing
* Jwt        -- Autheentication
* Heroku     -- Deployment

## Running and Installation of the app

```bash
$ npm install
$ npm start
```
